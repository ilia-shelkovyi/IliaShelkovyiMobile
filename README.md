# IliaShelkovyiMobile

To run native test:
mvn -Pnative clean test

To run web test:
mvn -Pweb clean test

To run cloud native test:
mvn -PcloudNative clean test

To run cloud web test:
mvn -PcloudWeb clean test

To run ios native test:
mvn -PiosNative clean test

To run ios web test:
mvn -PiosWeb clean test
